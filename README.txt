📚 ระบบบริหารจัดการแผนการจัดการเรียนรู้ออนไลน์ (Online Lesson Plan Management System)
ระบบบริหารจัดการแผนการจัดการเรียนรู้ที่พัฒนาขึ้นด้วย Google Apps Script ออกแบบมาเพื่อให้ครูผู้สอนสามารถส่งแผนการเรียนรู้ ตรวจสอบสถานะ และจัดการเอกสารในรูปแบบออนไลน์ได้อย่างสะดวกและรวดเร็ว โดยข้อมูลทั้งหมดจะถูกจัดเก็บไว้อย่างเป็นระบบบน Google Sheets และไฟล์เอกสารจะถูกจัดเก็บไว้ใน Google Drive
✨ คุณสมบัติเด่น (Features)
🔐 ระบบยืนยันตัวตน: รองรับการสมัครสมาชิกและล็อกอินแบ่งตามกลุ่มสาระการเรียนรู้
📝 ฟอร์มส่งงานออนไลน์: กรอกข้อมูลรหัสวิชา, ชื่อวิชา, หน่วยการเรียนรู้ และแผนการจัดการเรียนรู้ พร้อมอัปโหลดไฟล์ (PDF/DOCX)
📂 การจัดหมวดหมู่: รองรับกลุ่มสาระการเรียนรู้ทั้ง 8 กลุ่ม และระบบกรองข้อมูล (Filter) ตามสถานะ
📊 Dashboard & Stats: สรุปสถิติการส่งแผนการเรียนรู้ในรูปแบบแผนภูมิโดนัทและกราฟแท่ง
🖨️ พิมพ์ใบนำส่ง PDF: ระบบสร้างใบนำส่งที่มีรูปแบบถูกต้องสวยงาม สามารถพิมพ์หรือบันทึกเป็น PDF ได้ทันทีจากเบราว์เซอร์
🔔 ระบบแจ้งเตือนอัตโนมัติ:
แจ้งเตือนผ่าน Telegram ทันทีเมื่อมีครูส่งงานใหม่
แจ้งเตือนผ่าน Email เมื่อมีการอัปเดตสถานะ (ตรวจแล้ว/ให้แก้ไข)
📱 Responsive Design: ใช้งานได้สมบูรณ์ทั้งบนคอมพิวเตอร์ แท็บเล็ต และมือถือ
🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
Backend: Google Apps Script (Javascript V8)
Database: Google Sheets
Storage: Google Drive
Frontend: HTML5, CSS3, TailwindCSS
UI Components: SweetAlert2 (หน้าต่างแจ้งเตือน), Boxicons (ไอคอน)
Automation: GmailApp API, Telegram Bot API
🚀 ขั้นตอนการติดตั้ง (Installation & Setup)
เตรียมไฟล์:
สร้างไฟล์ Google Sheets ใหม่
สร้างโฟลเดอร์ใน Google Drive เพื่อไว้เก็บไฟล์แผนการเรียนรู้ (Copy Folder ID ไว้)
ตั้งค่า Google Apps Script:
ไปที่เมนู Extensions > Apps Script
นำโค้ดจากไฟล์ Code.gs ไปวาง และแก้ไข SPREADSHEET_ID และ FOLDER_ID ให้เป็นของท่าน
แก้ไข TELEGRAM_BOT_TOKEN และ TELEGRAM_CHAT_ID เพื่อเปิดใช้งานการแจ้งเตือน
เตรียมไฟล์ HTML:
สร้างไฟล์ HTML ใน Apps Script ชื่อ index1.html และนำโค้ดส่วนหน้าบ้านไปวาง
การ Deploy:
กดปุ่ม Deploy > New Deployment
เลือกประเภทเป็น Web App
ตั้งค่า "Who has access" เป็น Anyone (ทุกคน)
กดยืนยันสิทธิ์การเข้าถึงข้อมูล (Authorization)

หน้าแรก (Home)
